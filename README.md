# 🎬 Basic Animation

یک انیمیشن ساده و زیبا با HTML و CSS

## 📝 توضیحات پروژه
این پروژه یک انیمیشن ساده است که یک دایره را نشان می‌دهد که:
- از چپ به راست حرکت می‌کند
- رنگ آن از قرمز → آبی → سبز تغییر می‌کند
- سایز آن بزرگ و کوچک می‌شود

## 🚀 چگونه انیمیشن را ببینم؟
روی فایل `animation.html` کلیک کنید یا از لینک زیر استفاده کنید:
[مشاهده انیمیشن زنده](https://htmlpreview.github.io/?https://github.com/catxco/basic_animation/blob/main/animation.html)

## 🛠️ تکنولوژی‌ها
- HTML5
- CSS3 (Keyframes Animation)
